# CFARBEMCO-HRIS(CheckWise)

To install dependencies:

```bash
bun install
```
```bash
composer install
```

To run:

```bash
bun run build
```
```bash
bun run dev
```

To run:
```bash
php artisan serve
```

This project was created using `bun init` in bun v1.2.17. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
